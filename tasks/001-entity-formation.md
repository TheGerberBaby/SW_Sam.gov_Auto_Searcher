## Why this matters

Every cert and registration below it (SAM.gov, VetCert/SDVOSB,
FinCEN BOI, Virginia SWaM, eVA) requires a registered legal entity
with an EIN. Until this is `done`, every other task in this
directory is structurally blocked even if their status field says
otherwise.

- [ ] Operating agreement / bylaws in place?
- [ ] Registered agent in place?

If everything is already done, flip status to `done` and add a
note with the entity legal name + EIN's last four digits for
cross-reference.
